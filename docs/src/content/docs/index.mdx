## What is Kubelings?

**Kubelings** is hands-on Kubernetes practice in the spirit of *rustlings* and
*golings*: a sequence of small, broken-on-purpose clusters, each with an automated
check that only passes when you've genuinely fixed the problem.

Every scenario runs on iximiuz Labs **and** locally on `kind` from a single source
of truth — the [course](https://labs.iximiuz.com/courses/kubelings-dbd840c8).

The [nine-module curriculum](/guides/curriculum/) runs from first pod to
control-plane internals, with real cited production incidents — the
[Incident Library](/reference/incident-library/) — reproduced as runnable lessons
along the way.
